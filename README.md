HealthCare-Management-System
The HealthCare Management System is a Java-based application designed to simplify and automate hospital operations. It manages essential modules such as Patient Registration, Doctor Management, and Appointment Scheduling, ensuring efficient coordination between patients, doctors, and administrators. The system provides secure role-based access, allowing each user type to perform specific tasks and view relevant data.

Key Features: Patient Module: Register new patients, view doctor details, and book appointments. Doctor Module: View scheduled appointments and patient details. Admin Module: Manage doctors, patients, and appointment records. Database Integration: Uses MySQL with JDBC for storing and retrieving hospital data. Role-Based Login System: Ensures secure access control for each user type. User-Friendly Interface: Built with Java Swing for an intuitive desktop experience.

Tech Stack: Language: Java Database: MySQL Frameworks/Libraries: JDBC Architecture: MVC (Model-View-Controller) Pattern

Purpose: This project was developed to demonstrate efficient software design and database connectivity in Java, while addressing real-world hospital management needs like patient tracking and appointment handling.
